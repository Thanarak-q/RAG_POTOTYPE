# Privacy Policy Draft

This draft is for lawyer review before paid customer launch.

We process LINE chat messages to answer product questions, create sales leads when requested, and help shops manage customer conversations. We store LINE user IDs, message records, consent records, and lead records according to each shop's retention policy.

Customers may request deletion by messaging the bot with `delete my data` or by contacting the shop. Deletion requests remove chat sessions, messages, leads, and consent records associated with that LINE user ID unless law requires retention.

Subprocessors may include LINE, Vercel, Supabase, Pinecone, Upstash, and the selected LLM provider.
